# Sarvis Diligence

Acquisition diligence report for **Darling's Home Care** (Darling Apothecary, LLC), Warren, PA.

## Overview

An interactive single-page financial analysis dashboard covering:

- **Overview** — Business summary, revenue, profitability, and labor metrics
- **P&L Trend** — Revenue vs. operating income charts and key observations
- **Cost Structure** — Expense breakdown and year-over-year detail
- **Diligence Flags** — Red, amber, and green flags for the acquisition
- **Valuation** — Interactive EBITDA bridge and valuation matrix (3×–6× multiples)
- **Market** — Warren County demographics and regional market sizing

## Usage

Open `index.html` in any modern browser. No build step or dependencies required — Chart.js is loaded from CDN.

## Deployment

This site is deployed via [Vercel](https://vercel.com). Any push to `main` will trigger a redeployment.
